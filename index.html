TC on Cloud and Wireless Systems
for Industrial Applications

Welcome to the IEEE IES Technical Committee on Cloud and Wireless Systems for Industrial Applications (CWAIA) website!

This committee aims to provide a collaborative environment in which researchers and practitioners could work on identifying and showcasing ways in which emerging cloud and wireless technology together with related concepts, such as the Internet of Things, could be used to build new and improve existing industrial applications. The TSC aims to establish a network of experts and through regular events allow for dissemination of ideas and creation of new relationships.

In this web site you can find information about the activities undertaken by the Technical Committee. If you wish to be involved in these activities, join the CWAIA.
